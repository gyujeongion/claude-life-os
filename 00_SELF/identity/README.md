# identity/

The owner's self, in two files:

- **`character.md`** `[OWNER]` — who you are. You write this. The agent reads it before
  any taste or judgment call.
- **`character_ai_mirror.md`** `[AI]` — patterns the agent notices in you over time. The
  agent writes this; you review it.

Together they form a feedback loop: you declare who you are, the agent observes who you
actually are, and the gap between them is where the interesting conversations happen.

Keep both to what's *stable*. Fast-changing facts belong in `../memory/`, not here.
