# people/

`people.md` is the agent's memory of the humans in your life — only ones you've named or
confirmed. It's `[OWNER]`: the agent never fills it from guesses or web searches.

Why it matters: without it, the agent treats every name as new and either hallucinates
who they are or interrupts to ask every time. With it, "did you follow up with Sam?"
just works.

Keep clients and pure business contacts in their domain folders (`20_DOMAINS/…`), not
here. This file is for people, not accounts.
