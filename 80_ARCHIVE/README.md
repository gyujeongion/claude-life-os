# 80_ARCHIVE — done & inactive

Where finished projects and inactive material go to rest. Kept for reference, out of the
agent's active attention.

> 끝난 프로젝트와 비활성 자료가 쉬는 곳. 참고용으로 보관, 에이전트의 활성 주의에서 제외.

## Rules

- When a `10_WORK/` project hits `status: complete`, move the whole folder here.
- The agent does **not** treat archived material as current. It reads here only when you
  explicitly ask about past work.
- Keep an `INDEX.md` here if the archive grows, so old things stay findable.
